LEARN JAVASCRIPT

Данный источник очень сложный для новичка, поэтому рекомендую сначала пройти программу обучения от MDN.

Имея некоторый необходимый базис знаний о веб-разработке, поможет вам лучше понимать то, о чем говорится на Learn JS. При этом, поверьте! Будет трудно изучать данный ресурс! Но это не должно вас останавливать, пройденный путь того стоит!

Learn JS это учебник по языку программирования JavaScript и не только, доступный в двух форматах:

  • Веб-версия: LEARN JAVASCRIPT(бесплатная)
  • Версия PDF/EPUB книги (платная)

Learn JS состоит из 3 частей:

  • Язык JavaScript
  • Браузер: документ, события, интерфейсы
  • Тематические разделы (начинаются с - Фреймы и окна)

Если рассказывать о моём пути изучения этого источника, то это чтение по множеству раз одних и тех же тем, решение задач на которые бывало уходило пол дня, а то и несколько дней. На ознакомление с учебником я потратил примерно пол года и не жалею потраченного времени.

MDN(Mozilla Developer Network)

Ресурсы для разработчиков от разработчиков.

Данный источник содержит огромную базу информации по:

  • Веб-технологиям
  • CSS
  • HTML
  • JavaScript

MDN предоставляет учебную программу для веб-разработчиков. Данная программа бесплатная, нет временных рамок, отлично подходит для новичков! Присутствует перевод на русский язык.

На данный момент мне удалось освоить такие темы как:

  • Веб-стандарты
  • Семантика HTML
  • Основы CSS
  • Оформление текста CSS
  • CSS - макет

В будущем постараюсь освоить остальные темы.

Всем рекомендую ознакомиться с данным источником и обращаться к нему в своей работе.

Спасибо MDN за ваш труд!!!

Git and GitHub

Git — это система управления версиями, которая интеллектуально отслеживает изменения в файлах. Git особенно полезно, когда вы и группа людей вносят изменения в одни и те же файлы одновременно.

GitHub — это platform, где можно хранить, делиться и работать вместе с другими пользователями для написания кода.

Git and GitHub важные инструменты для разработчиков. Перед началом работы с ними рекомендую ознакомиться с документацией.

Хорошим помощником в понимании работы данных инструментов служит приложение LearnGitBranching.

freeCodeCamp

freeCodeCamp — это веб-приложение по обучению программирования.

Учебная программа состоит из прохождения курсов, после чего вы можете получить сертификат. Программа подходит для новичков, знание английского языка будет плюсом, т.к. к сожалению на данный момент нет русской версии приложения.

Рекомендую ознакомиться с приложением, думаю вы найдёте для себя новые интересные решения для своих работ.

Мой успех в данном приложении остановился на получении сертификата - Адаптивный веб-дизайн.
